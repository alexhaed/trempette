📋 Ton script Scriptable

  1. Installe l'app Scriptable (gratuite, App Store).
  2. Copie le script ci-dessous.
  3. Dans Scriptable : + → colle → renomme le script.
  4. Ajoute un widget Scriptable à ton écran d'accueil et choisis ce script.

    

      
    

    
Astuce : tu peux générer plusieurs scripts (un par plage) et créer plusieurs widgets.
